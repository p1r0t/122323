# 📱 Настройка Telegram группы для бота

## Зачем использовать группу?

✅ Несколько админов могут работать одновременно
✅ История всех операций в одном месте
✅ Удобнее для команды
✅ Можно назначать роли и права

## 🚀 Быстрая настройка

### Шаг 1: Создайте группу

1. Откройте Telegram
2. Нажмите "Новая группа"
3. Добавьте название (например: "Payment System Admin")
4. Добавьте участников (опционально)
5. Создайте группу

### Шаг 2: Добавьте бота в группу

1. Откройте вашу группу
2. Нажмите на название группы → "Добавить участников"
3. Найдите вашего бота по username (например: @YourBot)
4. Добавьте бота в группу

### Шаг 3: Сделайте бота администратором

1. Откройте группу → Название → "Администраторы"
2. Нажмите "Добавить администратора"
3. Выберите вашего бота
4. Включите права:
   - ✅ Удаление сообщений
   - ✅ Закрепление сообщений
   - ✅ Приглашение пользователей
5. Сохраните

### Шаг 4: Получите Chat ID группы

**Способ 1: Через бота @userinfobot**

1. Добавьте @userinfobot в вашу группу
2. Отправьте любое сообщение в группу
3. Бот ответит с информацией о группе
4. Скопируйте Chat ID (начинается с `-100`)
5. Удалите @userinfobot из группы

**Способ 2: Через API**

1. Отправьте любое сообщение в группу
2. Откройте в браузере:
   ```
   https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
   ```
3. Найдите `"chat":{"id":-1001234567890`
4. Скопируйте ID (с минусом!)

**Способ 3: Через команду**

```bash
curl "https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates" | grep -o '"chat":{"id":-[0-9]*' | head -1
```

### Шаг 5: Обновите config.php

```bash
sudo nano /var/www/payment/public_html/config.php
```

Замените CHAT_ID на ID вашей группы:

```php
define('CHAT_ID', '-1001234567890'); // ID группы с минусом!
```

### Шаг 6: Перезапустите бота (если используете polling)

```bash
sudo systemctl restart telegram-bot
```

Или для webhook:
```bash
# Webhook работает автоматически, ничего делать не нужно
```

## ✅ Проверка работы

### Тест отправки сообщения

```bash
curl -X POST "https://api.telegram.org/botYOUR_BOT_TOKEN/sendMessage" \
  -d "chat_id=-1001234567890" \
  -d "text=✅ Бот настроен и работает!"
```

Если сообщение пришло в группу - все работает!

## 🎯 Особенности работы в группе

### Inline кнопки работают для всех

Любой админ в группе может нажимать кнопки:
- 📱 Запросить OTP
- ❌ Ошибка
- ✅ Успех
- 💬 Чат

### Ответы в чат

Для ответа пользователю в чате отправьте сообщение в формате:

```
SESSION_ID Ваше сообщение
```

Или ответьте (Reply) на сообщение бота с Session ID.

### Уведомления

Все админы в группе получают уведомления о:
- Новых платежах
- Сообщениях от пользователей
- OTP кодах
- Активности в чате поддержки

## 🔧 Дополнительные настройки

### Отключить уведомления для определенных админов

1. Откройте группу
2. Нажмите на название → "Уведомления"
3. Настройте для себя

### Закрепить важное сообщение

Бот может закреплять сообщения, если у него есть права администратора.

### История сообщений

Все сообщения сохраняются в истории группы, что удобно для аудита.

## 🔒 Безопасность

### Рекомендации:

1. **Сделайте группу приватной**
   - Группа → Настройки → Тип группы → Приватная

2. **Ограничьте права участников**
   - Группа → Настройки → Права участников
   - Отключите "Отправка сообщений" для обычных участников

3. **Используйте 2FA для админов**
   - Настройки Telegram → Конфиденциальность → Двухэтапная аутентификация

4. **Регулярно проверяйте список участников**
   ```
   Группа → Участники → Проверьте всех
   ```

## ❓ FAQ

### Бот не отвечает в группе

**Проблема:** Бот не реагирует на команды

**Решение:**
1. Убедитесь что бот администратор
2. Проверьте права бота
3. Проверьте CHAT_ID в config.php (должен начинаться с `-100`)

### Кнопки не работают

**Проблема:** Inline кнопки не реагируют

**Решение:**
1. Бот должен быть администратором
2. Проверьте webhook: `curl "https://api.telegram.org/botYOUR_TOKEN/getWebhookInfo"`
3. Проверьте логи: `tail -f /var/www/payment/public_html/logs/bot_updates.log`

### Сообщения приходят дважды

**Проблема:** Дублирование уведомлений

**Решение:**
Проверьте что бот не добавлен в несколько групп с одинаковым CHAT_ID.

### Как добавить второго админа?

1. Откройте группу
2. Название → "Администраторы"
3. "Добавить администратора"
4. Выберите пользователя
5. Настройте права
